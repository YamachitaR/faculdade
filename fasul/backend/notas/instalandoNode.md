# Verificando se tem instalado


Instale `node` e `npm` 

Verifique se realmente foi instalado 
~~~
node --version 
npm –version
~~~

##  Express

- Express.js é um framework leve, rápido e básico para Node.js. Ele fornece uma das maneiras mais poderosas e fáceis de criar um servidor web.

- Um servidor é simplesmente um computador que escuta as solicitações recebidas.
